# Word Search Game

Word Search 게임을 생성하고 플레이할 수 있는 웹 애플리케이션입니다.

## 기능

- Word Search 게임 생성 (Word Search Maker)
- 생성된 게임 플레이
- 실시간 현황판
- 사용자 인증 및 점수 기록

## 기술 스택

### 백엔드
- Python
- FastAPI
- SQLAlchemy (ORM)
- WebSockets
- SQLite/MySQL

### 프론트엔드
- HTML/CSS/JavaScript
- Svelte
